# 🌿 CapyFinance | Wealth Management

**CapyFinance** is a premium wealth management platform inspired by the rhythmic precision of the Pantanal ecosystem. Built with React and designed for a high-end editorial experience, it combines powerful financial toolsets with a calm, organic aesthetic.

![CapyFinance Dashboard Mockup](https://images.unsplash.com/photo-1590424753858-3b6b1f31b6df?ixlib=rb-1.2.1&auto=format&fit=crop&w=1200&q=80)

## ✨ Features

- **Premium Design System**: Glassmorphism, tonal depth, and custom Pantanal-inspired color palettes.
- **Multilingual Support**: Fully localized in English, Portuguese, Spanish, French, Italian, Chinese, Japanese, Korean, and Arabic.
- **RTL Compatibility**: Automatic layout mirroring for Right-to-Left languages (Arabic).
- **Core Modules**:
  - **Login**: Secure, editorial-style access.
  - **Dashboard**: Real-time market indices (Ibovespa, Selic) and portfolio ecosystem tracking.
  - **Valuation**: Strategic company evaluation tool with growth projection models.
  - **Education**: Curated financial paths for sustainable wealth cultivation.

## 🚀 Tech Stack

- [React](https://reactjs.org/) (Vite)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Router DOM](https://reactrouter.com/)
- [i18next](https://www.i18next.com/) & `react-i18next`
- [Framer Motion](https://www.framer.com/motion/) (Animations)

## 💻 Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/erianedias/SiteEriane.git
   cd SiteEriane
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

## 🌐 Deployment to Vercel

### Option 1: Vercel Dashboard (Recommended)

1. Push your code to a GitHub/GitLab/Bitbucket repository.
2. Sign in to [Vercel](https://vercel.com).
3. Click "New Project" and import your repository.
4. Vercel will automatically detect **Vite** as the framework.
5. Click **Deploy**.

### Option 2: Vercel CLI

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```
2. Run the deployment command in the project root:
   ```bash
   vercel
   ```

## 🎨 Design Principles (The Riparian Flow)

- **Organic Precision**: Marrying reliability with fluid, nature-inspired elements.
- **The "No-Line" Rule**: Boundaries are defined through background shifts, not borders.
- **Tonal Layering**: Hierarchy is established through physical layers and elevation.
